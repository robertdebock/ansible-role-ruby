ansible-role-ruby
=========

Provides Ruby for your system.

Requirements
------------

Access to a repository containing packages, likely on the internet.

Role Variables
--------------

None known.

Dependencies
------------

- robertdebock.ansible-role-bootstrap

Example Playbook
----------------

```
- hosts: servers

  roles:
    - robertdebock.ansible-role-ruby
```
